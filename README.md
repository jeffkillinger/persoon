# Persoon Personalization Sandbox

A React + Next.js demo that simulates a rule-based personalization engine for SaaS onboarding and marketing experiences.

This project demonstrates:

• React state-driven UI
• TypeScript domain modeling
• Rule evaluation logic
• Real-time preview updates
• Tailwind dashboard UI

## Live Demo

https://persoon-demo.vercel.app

## Run Locally

npm install
npm run dev
